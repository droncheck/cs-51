# Схема кодирования символов в биты

## 1. Общая структура кода

Код символа всегда начинается с **2-битного префикса**, затем идёт **суффикс** фиксированной для ветки длины:

| Префикс (2 бита) | Категория                                | Длина суффикса | Всего бит на символ |
| ---------------- | ---------------------------------------- | -------------- | ------------------- |
| `00`             | Цифры, управляющие символы, `Ё` и `ё`    | 4              | 6                   |
| `10`             | Заглавные буквы (кириллица, без `Ё`/`ё`) | 5              | 7                   |
| `01`             | Строчные буквы (кириллица, без `Ё`/`ё`)  | 5              | 7                   |
| `11`             | Знаки препинания и специальные символы   | 5              | 7                   |

---

## 2. Ветка `00` — суффикс 4 бита (16 значений: 0–15)

Подсуффикс задаёт строку в таблице ниже.

| №   | Двоичный суффикс | Символ                               |
| --- | ---------------- | ------------------------------------ |
| 0   | `0000`           | `0`                                  |
| 1   | `0001`           | `1`                                  |
| 2   | `0010`           | `2`                                  |
| 3   | `0011`           | `3`                                  |
| 4   | `0100`           | `4`                                  |
| 5   | `0101`           | `5`                                  |
| 6   | `0110`           | `6`                                  |
| 7   | `0111`           | `7`                                  |
| 8   | `1000`           | `8`                                  |
| 9   | `1001`           | `9`                                  |
| 10  | `1010`           | `Ё`                                  |
| 11  | `1011`           | `ё`                                  |
| 12  | `1100`           | пробел (U+0020)                      |
| 13  | `1101`           | горизонтальная табуляция HT (U+0009) |
| 14  | `1110`           | перевод строки LF (U+000A)           |
| 15  | `1111`           | возврат каретки CR (U+000D)          |

**Полный код ветки `00`:** биты `00` + двоичная запись значения 0–15 в **4 битах** (с ведущими нулями при необходимости).

---

## 3. Ветки `10` и `01` — суффикс 5 бит (32 буквы)

Используются **32 буквы** русского алфавита **без** `Ё` и `ё`, в **традиционном алфавитном порядке** (как в школьной азбуке, буква `Е` без отдельной `Ё`):

`А Б В Г Д Е Ж З И Й К Л М Н О П Р С Т У Ф Х Ц Ч Ш Щ Ъ Ы Ь Э Ю Я`

Суффикс — номер буквы в этом списке **с нуля** (0 = А, 31 = Я), в **5 битах** (0…31).

### 3.1. Заглавные буквы — префикс `10`

| №   | Буква | Двоичный суффикс |
| --- | ----- | ---------------- |
| 0   | А     | `00000`          |
| 1   | Б     | `00001`          |
| 2   | В     | `00010`          |
| 3   | Г     | `00011`          |
| 4   | Д     | `00100`          |
| 5   | Е     | `00101`          |
| 6   | Ж     | `00110`          |
| 7   | З     | `00111`          |
| 8   | И     | `01000`          |
| 9   | Й     | `01001`          |
| 10  | К     | `01010`          |
| 11  | Л     | `01011`          |
| 12  | М     | `01100`          |
| 13  | Н     | `01101`          |
| 14  | О     | `01110`          |
| 15  | П     | `01111`          |
| 16  | Р     | `10000`          |
| 17  | С     | `10001`          |
| 18  | Т     | `10010`          |
| 19  | У     | `10011`          |
| 20  | Ф     | `10100`          |
| 21  | Х     | `10101`          |
| 22  | Ц     | `10110`          |
| 23  | Ч     | `10111`          |
| 24  | Ш     | `11000`          |
| 25  | Щ     | `11001`          |
| 26  | Ъ     | `11010`          |
| 27  | Ы     | `11011`          |
| 28  | Ь     | `11100`          |
| 29  | Э     | `11101`          |
| 30  | Ю     | `11110`          |
| 31  | Я     | `11111`          |

### 3.2. Строчные буквы — префикс `01`

Тот же порядок номеров 0…31, те же суффиксы, но символы — **строчные**:

`а б в г д е ж з и й к л м н о п р с т у ф х ц ч ш щ ъ ы ь э ю я`

Пример: буква **м** — это № 12 в списке, суффикс `01100`, полный код начинается с `01`: `01` + `01100`.

## 4. Ветка `11` — знаки препинания и специальные символы, суффикс 5 бит

Задано **31 знак**. Коды суффикса **0–30** соответствуют символам в таблице; значение суффикса **31** (`11111`) **не используется** (резерв).

| №   | Двоичный суффикс | Символ            |
| --- | ---------------- | ----------------- |
| 0   | `00000`          | `.`               |
| 1   | `00001`          | `,`               |
| 2   | `00010`          | `;`               |
| 3   | `00011`          | `:`               |
| 4   | `00100`          | `!`               |
| 5   | `00101`          | `?`               |
| 6   | `00110`          | `«`               |
| 7   | `00111`          | `»`               |
| 8   | `01000`          | `—`               |
| 9   | `01001`          | `-`               |
| 10  | `01010`          | `(`               |
| 11  | `01011`          | `)`               |
| 12  | `01100`          | `…`               |
| 13  | `01101`          | `"`               |
| 14  | `01110`          | `'`               |
| 15  | `01111`          | `№`               |
| 16  | `10000`          | `%`               |
| 17  | `10001`          | `*`               |
| 18  | `10010`          | `+`               |
| 19  | `10011`          | `=`               |
| 20  | `10100`          | `<`               |
| 21  | `10101`          | `>`               |
| 22  | `10110`          | `@`               |
| 23  | `10111`          | `#`               |
| 24  | `11000`          | `&`               |
| 25  | `11001`          | `[`               |
| 26  | `11010`          | `]`               |
| 27  | `11011`          | `{`               |
| 28  | `11100`          | `}`               |
| 29  | `11101`          | `/`               |
| 30  | `11110`          | `\`               |
| —   | `11111`          | _не используется_ |

**Полный код ветки `11`:** биты `11` + 5 бит суффикса по таблице.
